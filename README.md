# Python Practice Codes

This repository contains a collection of Python practice codes organized by topics and days. It is designed to help improve coding skills through hands-on exercises and mini-projects.

## Structure

The repository is organized as follows:

- **python/**: Contains Python practice codes organized by days and topics:
  - **Day1/**: Basic Python concepts and introductory exercises.
  - **Day2/**: Problem-solving exercises on various topics.
  - **Day3/**: Advanced topics including:
    - **decorators/**: Timing functions, debugging, and caching.
    - **functions/**: Recursive functions, lambda expressions, and argument handling.
    - **loops/**: Prime numbers, factorial calculation, and unique lists.
    - **oops/**: Object-oriented programming concepts.
  - **Day4/**: Miscellaneous basic codes.
  - **Day5/**: Mini projects and utility scripts, including crew management and data handling.
- **javascript/**: Placeholder for JavaScript practice codes.
- **react/**: Placeholder for React-related projects.
- **sql/**: SQL queries and database exercises.

## Topics Covered

- Python Basics
- Data Types
- Loops and Functions
- Object-Oriented Programming
- Decorators
- Mini Projects
- SQL Queries

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Blaze99git/practice_codes.git
   ```
2. Navigate to the desired folder and explore the scripts.
3. Run the Python scripts using Python 3.x:
   ```bash
   python script_name.py
   ```
